Instructions

Drag the variable names from the first column to the column/row to the right of it to select the variables whose cross tabulation you want to compute. Note that you can drag multiple variables to create a multilevel variable. Use the top dropdown menu to change the visualization style, and the menu below it to change the aggregation function.

Notes

  • All figures are in 2016 dollars.
  • The full data set is available for download from the Federal Reserve Board's website.
  • Due to the survey design, the row weights are not integers, and thus it is possible to get fractional results for features that are usually integral.
  • The aggregator Count returns the weighted count, i.e. the estimated count in the population. The aggregator Sample Count gives the count in the sample before weighting/replication.
